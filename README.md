# Sudoku

# How to run
- Edit the register.bat (notepad)
- Define the path to the folder on your computer
- Run as admin
- Go to Visual Studio
- - Right click on "Sa-Week4-Sudoku"
- - Add -> reference
- - Com -> Search "Sudoku"
- - Make sure "Sudoku Game" is on
- - Save.

You're now ready to run the application.
